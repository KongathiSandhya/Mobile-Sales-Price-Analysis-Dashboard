An interactive data analytics dashboard for exploring mobile phone pricing trends, brand comparisons, and sales insights — built with Power BI / Tableau.

## Overview
This dashboard provides a comprehensive analysis of mobile phone pricing and sales data across multiple brands and product categories. It transforms raw sales data into meaningful visual insights, helping stakeholders identify pricing patterns, compare brand performance, and understand market trends at a glance.

## Key Features
- 📈 Price trend analysis across brands and models
- 🔍 Dynamic filters for brand, price range, and category
- 📊 KPI cards for average price, sales volume, and top performers
- 🔄 Interactive drill-downs for in-depth exploration

## Tools & Technologies
- Power BI / Tableau — dashboard design and visualization
- Microsoft Excel / CSV — data source and preprocessing
- DAX / Calculated Fields — custom KPIs and measures

## How to Use
1. Open the `.pbix` / `.twbx` file in Power BI Desktop / Tableau
2. Use the slicers to filter by brand, model, or price range
3. Hover over visuals for detailed tooltips and breakdowns
